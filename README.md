 🧮 SAIT Theme Changer

A simple web application built with **HTML, CSS, and JavaScript** that allows users to switch between different themes dynamically.

 ✨ Features
- Switch between multiple themes
- Interactive user interface
- DOM manipulation using JavaScript

 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

 📚 About This Project
This project was developed as part of my training at the **School of Artificial Intelligence and Technology (SAIT)** to practice JavaScript fundamentals and DOM manipulation.
